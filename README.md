# RevStaff Metrics Dashboard

2025 SQL funnel performance analysis for RevStaff leadership.

## Charts Included

- **View 1**: SQL Volume by Source by Month
- **View 2**: Rep Performance Distributions (Total, Prospecting+Conference, Per-Rep)
- **View 2.3-2.4**: Account Tier Analysis (Tier 1 vs Non-Tier 1)
- **View 2.5-2.8**: Source-specific (Conference, Demo Request, Referrals)

## Data

- Source: HubSpot export Dec 29, 2025
- Total: 416 SQLs
- Methodology: Cohort Month (when deal was created)

pip install streamlit
